Acerca del Proyecto

Este sistema web tiene como objetivo principal administrar eficientemente la asignación de aulas tanto a Auxiliares como a Docentes en la carrera de Ingeniería de Sistemas de la UPEA. Proporciona información precisa, oportuna y confiable para facilitar la generación automática de consultas y reportes requeridos por el Director de Carrera y el personal administrativo.

Leer más

Objetivos Específicos

  • Realizar el levantamiento de requerimientos y análisis para la organización y asignación de aulas, incluyendo datos históricos y capacidad de las aulas.
  • Implementar una programación de aulas desde cero basada en parámetros asignados por el personal administrativo.
  • Generar informes detallados sobre el nivel de uso de los recursos físicos, incluyendo el estado y capacidad de cada aula.
